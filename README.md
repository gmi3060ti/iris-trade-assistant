# IRIS Trade Assistant

IRIS Trade Assistant is a desktop application that assists with chart analysis and trade decision support.

## Features

- Modern desktop GUI
- Screen capture
- Technical analysis
- AI scoring
- Trade notifications
- Logging system
- Configuration management
